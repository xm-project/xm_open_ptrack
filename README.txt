# xm_open_ptrack

Create Date: 2015.11.1

Authors: OpenPTrack, frame_of_youth

Function:
OpenPTrack is an open source and successful project launched in 2013 to create a scalable, multi-camera solution for person tracking. It enables many people to be tracked over large areas in real time. Our xmbot team use openptrack package to develop our people follow test.
